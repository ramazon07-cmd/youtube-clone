# UzTube

UzTube is a localized video-sharing platform focused on Uzbekistan. It is a rebranded YouTube clone with Uzbek-first UX, regional discovery, and multilingual support (O'zbek, Russian, English).

## Project Description

UzTube allows users to:
- browse and watch uploaded videos
- upload videos with optional custom thumbnails
- like/dislike content
- view channel-based video lists
- explore local categories and trending videos in Uzbekistan

## Tech Stack

- **Backend:** Django
- **Templates:** Django Templates (server-rendered)
- **Frontend:** HTML, CSS, Vanilla JavaScript
- **Media Pipeline:** ImageKit integration for video/thumbnail handling
